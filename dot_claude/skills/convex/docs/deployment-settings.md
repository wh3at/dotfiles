On this page

The [deployment settings page](https://dashboard.convex.dev/deployment/settings)
gives you access to information and configuration options related to a specific
deployment (**production**, your personal **development** deployment, or a
**preview** deployment).

## URL and Deploy Key[​](https://docs.convex.dev/dashboard/deployments/deployment-settings.html#url-and-deploy-key "Direct link to URL and Deploy Key")

The [URL and deploy key page](https://dashboard.convex.dev/deployment/settings)
shows:

* The URL this deployment is hosted at. Some Convex integrations may require the
  deployment URL for configuration.
* The URL that HTTP Actions for this deployment should be sent to.
* The deployment's deploy key, used to
  [integrate with build tools such as Netlify and Vercel](https://docs.convex.dev/production/hosting/index.html)
  and
  [syncing data with Fivetran and Airbyte](https://docs.convex.dev/production/integrations/streaming-import-export.html).

![Deployment Settings Dashboard Page](https://docs.convex.dev/assets/images/deployment_settings-58661797d5cadbc484d3d36dde845c04.png)

## Environment Variables[​](https://docs.convex.dev/dashboard/deployments/deployment-settings.html#environment-variables "Direct link to Environment Variables")

The
[environment variables page](https://dashboard.convex.dev/deployment/settings/environment-variables)
lets you add, change, remove and copy the deployment's
[environment variables](https://docs.convex.dev/production/environment-variables.html).

![deployment settings environment variables page](https://docs.convex.dev/assets/images/deployment_settings_env_vars-e148766325f85db3409292b10f202bba.png)

## Authentication[​](https://docs.convex.dev/dashboard/deployments/deployment-settings.html#authentication "Direct link to Authentication")

The
[authentication page](https://dashboard.convex.dev/deployment/settings/authentication)
shows the values configured in your `auth.config.js` for user
[authentication](https://docs.convex.dev/auth.html) implementation.

## Backup & Restore[​](https://docs.convex.dev/dashboard/deployments/deployment-settings.html#backup--restore "Direct link to Backup & Restore")

The
[backup & restore page](https://dashboard.convex.dev/deployment/settings/backups)
lets you [backup](https://docs.convex.dev/database/backup-restore.html) the data stored in your
deployment's database and file storage. On this page, you can schedule periodic
backups.

![deployment settings export page](https://docs.convex.dev/assets/images/backups-7e17da1541fc3eb26194a96ab33414ea.png)

## Integrations[​](https://docs.convex.dev/dashboard/deployments/deployment-settings.html#integrations "Direct link to Integrations")

The integrations page allows you to configure
[log streaming](https://docs.convex.dev/production/integrations/index.html),
[exception reporting](https://docs.convex.dev/production/integrations/index.html), and
[streaming export](https://docs.convex.dev/production/integrations/streaming-import-export.html)
integrations.

## Pause Deployment[​](https://docs.convex.dev/dashboard/deployments/deployment-settings.html#pause-deployment "Direct link to Pause Deployment")

On the
[pause deployment page](https://dashboard.convex.dev/deployment/settings/pause-deployment)
you can [pause your deployment](https://docs.convex.dev/production/pause-deployment.html) with the pause
button.

![deployment settings pause deployment page](https://docs.convex.dev/assets/images/deployment_settings_pause-4e036413269bccced2d58a99d2bb6f98.png)
