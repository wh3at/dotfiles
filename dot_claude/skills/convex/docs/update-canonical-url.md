# Update canonical URL

```
POST

## /update_canonical_url
```

Set or unset the canonical URL for a deployment's convex.cloud or
convex.site domain. This allows you to customize the CONVEX\_SITE\_URL and
CONVEX\_CLOUD\_URL environment variables in your deployment.

## Request[​](https://docs.convex.dev/deployment-api/update-canonical-url.html#request "Direct link to Request")

## Responses[​](https://docs.convex.dev/deployment-api/update-canonical-url.html#responses "Direct link to Responses")

* 200
