You can export your data from Convex by
[taking a backup](https://docs.convex.dev/database/backup-restore.html) and downloading it as a zip file.

Alternatively, you can export the same data with the
[command line](https://docs.convex.dev/using/cli.html#export-data-to-a-file):

```
npx convex export --path ~/Downloads
```
