# Function mixedCase

* mixedCase(
      stringArb: [Arbitrary](https://fast-check.dev/api-reference/classes/Arbitrary.html)<string>,
      constraints?: [MixedCaseConstraints](https://fast-check.dev/api-reference/interfaces/MixedCaseConstraints.html),
  ): [Arbitrary](https://fast-check.dev/api-reference/classes/Arbitrary.html)<string>

  Randomly switch the case of characters generated by `stringArb` (upper/lower)

  WARNING:
  Require bigint support.
  Under-the-hood the arbitrary relies on bigint to compute the flags that should be toggled or not.

  #### Parameters

  + stringArb: [Arbitrary](https://fast-check.dev/api-reference/classes/Arbitrary.html)<string>

    Arbitrary able to build string values
  + `Optional`constraints: [MixedCaseConstraints](https://fast-check.dev/api-reference/interfaces/MixedCaseConstraints.html)

    Constraints to be applied when computing upper/lower case version
