# Variable toStringMethod`Const`

toStringMethod: unique symbol = ...

Use this symbol to define a custom serializer for your instances.
Serializer must be a function returning a string (see [WithToStringMethod](https://fast-check.dev/api-reference/types/WithToStringMethod.html)).
