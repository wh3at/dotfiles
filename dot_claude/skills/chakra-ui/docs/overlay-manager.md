# Overlay Manager

Used for programmatically controlling overlay components

AI TipWant to skip the docs? Use the [MCP Server](https://chakra-ui.com/docs/get-started/ai/mcp-server)

Copy Page

## [Usage](https://chakra-ui.com/docs/components/overlay-manager.html#usage)

The `createOverlay` function creates a new overlay component that can be
programmatically controlled.

```
import { createOverlay } from "@chakra-ui/react"

const dialog = createOverlay<DialogProps>((props) => {
  const { title, description, content, ...rest } = props
  return (
    <Dialog.Root {...rest}>
      <Portal>
        <Dialog.Backdrop />
        <Dialog.Positioner>
          <Dialog.Content>
            {title && (
              <Dialog.Header>
                <Dialog.Title>{title}</Dialog.Title>
              </Dialog.Header>
            )}
            <Dialog.Body spaceY="4">
              {description && (
                <Dialog.Description>{description}</Dialog.Description>
              )}
              {content}
            </Dialog.Body>
          </Dialog.Content>
        </Dialog.Positioner>
      </Portal>
    </Dialog.Root>
  )
})
```

Then render the `Viewport` component to see the overlay.

```
<dialog.Viewport />
```

### [Programmatic Closing](https://chakra-ui.com/docs/components/overlay-manager.html#programmatic-closing)

You can close an overlay from within the component itself by calling the
injected `onOpenChange` prop with `{ open: false }`.

This is useful for scenarios like form submissions or user interactions that
should close the overlay.

```
const dialog = createOverlay<DialogProps>((props) => {
  const { onOpenChange, ...rest } = props

  const handleSubmit = () => {
    // Close the overlay after successful action
    onOpenChange?.({ open: false })
  }

  return <Dialog.Root {...rest}>{/* ... */}</Dialog.Root>
})
```

## [API](https://chakra-ui.com/docs/components/overlay-manager.html#api)

### [Props](https://chakra-ui.com/docs/components/overlay-manager.html#props)

Props that are injected into the overlay component by the `createOverlay`
function:

* `open`: Whether the overlay is currently open
* `onOpenChange`: Callback fired when the overlay's open state changes
* `onExitComplete`: Callback fired when the overlay's exit animation completes

### [Methods](https://chakra-ui.com/docs/components/overlay-manager.html#methods)

### [`Viewport`](https://chakra-ui.com/docs/components/overlay-manager.html#viewport)

The root component that renders all active overlays.

### [`open(id, props)`](https://chakra-ui.com/docs/components/overlay-manager.html#openid-props)

Opens a new overlay with the given id and props. Returns a promise that resolves
with any value.

### [`close(id, value)`](https://chakra-ui.com/docs/components/overlay-manager.html#closeid-value)

Closes the overlay with the given id and returns a promise that resolves when
closed.

### [`update(id, props)`](https://chakra-ui.com/docs/components/overlay-manager.html#updateid-props)

Updates the props of the overlay with the given id.

### [`remove(id)`](https://chakra-ui.com/docs/components/overlay-manager.html#removeid)

Removes the overlay with the given id.

### [`removeAll()`](https://chakra-ui.com/docs/components/overlay-manager.html#removeall)

Removes all overlays.

### [`get(id)`](https://chakra-ui.com/docs/components/overlay-manager.html#getid)

Gets the props of the overlay with the given id.

### [`getSnapshot()`](https://chakra-ui.com/docs/components/overlay-manager.html#getsnapshot)

Gets the current snapshot of the overlays.

### [`waitForExit(id)`](https://chakra-ui.com/docs/components/overlay-manager.html#waitforexitid)

Waits for the exit animation to complete for the overlay with the given id.
